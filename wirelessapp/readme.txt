RemotePowerPoint App instructions:

1.Unzip the file and open to find two separate projects to be opened in eclipse and android studio.

2. Open RemoteBluetoothServer project in eclipse and specify the powerpoint file name with full location in .ppt format in the ProcessConnectionThread.java file.

3. Open RemoteBluetooth project in Android Studio.

4. Run the Remote RemoteBluetoothServer.java file to start the bluetooth server.

5. Build the application in android studio using build apk which shows the path to the .apk file.

6. Copy the file to android device and open it to install on the device.

7. In the options menu press connect a device to connect the android phone to pc

8. Open the presentation in your pc and you can see the application to perform required functions.
